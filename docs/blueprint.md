# **App Name**: Firestore Fortress

## Core Features:

- Analytics Data Collection: Collect and store analytics data with Firestore based on the provided security rules.
- Academic Project Management: Admin-controlled management of academic projects with role-based access.
- Role Validation: Generative AI powered role validation tool that is used to check Firestore document accesses based on user roles and project access levels.
- Secure Data Access: Enforce Firestore security rules for all read and write operations.
- UI Dashboard: Create and display user data in a clean and secure user interface

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5), suggesting security and depth.
- Background color: Light Indigo (#E8EAF6), a desaturated, bright tint of the primary color for a calm feel.
- Accent color: Blue (#3F51B5) analogous to indigo, used to highlight important actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for both headlines and body text.
- Use consistent, professional icons related to data security and project management.
- Design a clean and intuitive layout with clear data visualization for analytics and project statuses.
- Incorporate subtle animations for loading states and transitions to enhance the user experience without distraction.
